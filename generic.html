Project Discriptions

  • Java Marksheet Program

    The Java Marksheet Program is a robust application designed to manage and display student academic records. It allows users to input and store student names, roll numbers, and subject marks, and then calculates and displays total marks, percentage, and grades. This project showcases my proficiency in Java's object-oriented principles and my ability to create user-friendly interfaces. It effectively demonstrates my skills in handling user input, performing calculations, and presenting data in a structured format.

  • Calculator using c

    This C calculator extends traditional functionality by incorporating a modulus operation, enhancing its utility for various mathematical computations. The project emphasizes my understanding of fundamental programming concepts in C, including operators, conditionals, and loops. The calculator supports basic arithmetic operations—addition, subtraction, multiplication, division, and modulus—providing a comprehensive tool for mathematical problem-solving. Its implementation reflects my attention to detail and commitment to creating functional, user-oriented software solutions.

  • C++ Tic-Tac-Toe Game

    The C++ Tic-Tac-Toe Game is an interactive console application that allows two players to compete in the classic game of Tic-Tac-Toe. The project highlights my skills in C++ programming, including the use of arrays, loops, and functions to manage game logic and user input. It features a simple, intuitive interface and includes functionality for detecting win conditions and handling invalid moves. This project demonstrates my ability to design and implement game mechanics and manage program state effectively.
